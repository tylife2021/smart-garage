# ESP32 Smart Controller

A production-style Flutter app for controlling and monitoring an ESP32-based device.

## Included
- Clean architecture with MVVM + Riverpod
- REST, WebSocket, and MQTT communication abstractions
- Home screen, sensor dashboard, WiFi setup, and settings
- Example ESP32 Arduino sketch exposing REST endpoints

## Structure
- lib/core
- lib/providers
- lib/repositories
- lib/screens
- lib/services
- lib/theme
- lib/routes

## Next steps
1. Install Flutter SDK.
2. Run `flutter pub get`.
3. Run `flutter run`.
